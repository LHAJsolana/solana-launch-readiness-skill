- Type: Solana consumer dApp with a small SPL rewards token
- Target network: mainnet
- Target launch date: 2026-07-15

- Monitoring is not fully configured.

- Blockers and high-priority fixes.

- Wallet UX checklist.
- Token launch risks.
- Incident response plan gaps.
