# Rule: Safe Mainnet Behavior

- Default to devnet/localnet for examples and scripts.
- Warn before mainnet actions.
- Require explicit human confirmation for irreversible operations.
- Never suggest deceptive token launches, fake hype, fake partnerships, fake TVL, wash trading, or market manipulation.
- Never claim legal, financial, tax, investment, or formal audit certainty.
- Make mainnet configuration explicit: cluster, RPC URL, program ID, token mint, authority, and signer path.
- Prefer dry-runs, simulation, and manual review before mainnet deployment or authority changes.
